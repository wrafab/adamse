I revieved the template for my box from website called Maker Case, then began designing it in Corel Draw. I wanted to make my box rectangular so I made the dimentions 3x6 inches. I then imported pictures I wanted engraved on the sides, then made sure the border was set to hariline so it would raster and cut completly through the cardboard. I used the Epilog Fusion 5W M2 printer to print my box and I am satisfied with how it turend out. Overall this project was prtty easy and I think that it is so cool that we can print anything on almost any material using this printer.

Box Design


I designed my three shapes in Croel Draw and then placed small rectablges along the inside of the shapes in order to make them lock with eachother. I made all of the boarders for all of the shapes hariline then printed it in the the Epilog Fusion 5W M2 printer. All of the shapes fit together niceley which was my goal and I'm happy that it turned out the way it was supposed to.
