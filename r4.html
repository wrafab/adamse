I recieved the template for my box from website called Maker Case, then began designing it in Corel Draw. I wanted to make my box rectangular so I made the dimentions 3x6 inches. I then imported pictures I wanted engraved on the sides, then made sure the border was set to hariline so it would raster and cut completly through the cardboard. I used the Epilog Fusion 5W M2 printer to print my box then used hot glue to fit it together. Overall this project was prtty easy and I think that it is so cool that we can print anything on almost any material using this printer.

Box Design


























I designed my three shapes in Corel Draw and then placed small rectanlges along the inside of the shapes in order to make them lock with eachother. I made all of the boarders for all of the shapes hariline then printed it in the the Epilog Fusion 5W M2 printer. All of the shapes fit together niceley which was my goal and I'm happy that it turned out the way it was intended.

Press Fit Kit Design





















































Next I chose to design a glass for my dads birthday. I used and inmage of the Ohio EPA's logo and put it Corel Draw, resized it then printed it using the Epilog Fusion 5W M2 printer. When I enlarged the image the edges got a little blurry which then translated into the etching. Other than that small mistake I am happy with how the glass turned out and excited to see how my dad will like it.

Glass Design






























Lastly I made a wristband that I once again designed using Corel Draw. I put one of my favoirte words, "wabi-sabi" which is a world view centered on the acceptance of transience and imperfection. I used the Epilog Fusion 5W M2 printer and I had a difficult time orienting the design so it would prit in the middle of the wristband, so that would be the only thing I would change.
